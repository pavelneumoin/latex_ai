# cli/ — спринт-папка к 2026-05-11

Цель — за 6 дней собрать минимальную CLI-цепочку «PDF → класс → фото → проверка», чтобы протестировать чекер на реальном классе пользователя.

**Тест-сценарий:** ЕГЭ-профиль №10 (текстовые задачи), класс 11, 5 уроков 2026-05-11 → 5 разных шаблонов с разными прототипами:

| Шаблон | Прототип | Источник в банке `Lessons/vault/02-Банк_задач/ЕГЭ_мат_mathege/10_текстовые_задачи/` |
|--------|----------|---------------------------------------|
| T1 | Движение по реке | `text_motion_river` |
| T2 | Совместная работа (трубы) | подтипы про трубы/насосы |
| T3 | Концентрация / смеси | сплавы и растворы |
| T4 | Проценты (двойное изменение) | подорожали-подешевели |
| T5 | Средняя скорость туда-обратно | средняя скорость |

**Важно:** на этом спринте задачи берутся из готового банка (не LLM-генерация). Это убирает риск «LLM нагенерила бред» — банк уже проверен. LLM подключаем после теста чекера, на этапе сайта.

## Папки

```
cli/
├── templates/         5 LaTeX-шаблонов (T1..T5)
├── prompts/           LLM-промпты под генерацию задач и vision-проверку
├── output/            сгенерированные PDF + answer_key.json
├── test_photos/       фото от тестового класса (по папкам на ученика)
└── (скрипты в корне)
```

## Скрипты (создаются по дням)

- `generate.py` — Day 3
- `check.py` — Day 4
- `extract_answer_fields.py` — Day 2

## Поток

```
[generate.py] → output/{lesson_id}.pdf
              → output/{lesson_id}.answer_key.json
                                                       ↓ печать → класс → фото
                                                       ↓
[check.py answer_key.json photos/*.jpg] → output/{lesson_id}.results.json
```

## Формат `answer_key.json`

```json
{
  "lesson_id": "L1-derivative-2026-05-11",
  "template": "T1",
  "topic": "Производная сложной функции",
  "grade": 11,
  "subject": "math",
  "fields": [
    {
      "task_id": 1,
      "page": 1,
      "x_mm": 150,
      "y_mm": 250,
      "width_mm": 30,
      "height_mm": 12,
      "expected": "42",
      "answer_type": "number",
      "tolerance": 0.01
    }
  ]
}
```

Координаты — от верхнего левого угла страницы (LaTeX-стандарт через `zref-savepos`).
