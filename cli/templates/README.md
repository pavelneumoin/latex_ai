# WorksheetAI — система LaTeX-шаблонов

В этой папке лежат 15 шаблонов рабочих листов (`T1`…`T15`), каждый в своей подпапке (`T<N>/T<N>.tex`). Шаблоны отличаются раскладкой (1 колонка, 2 колонки, плитка-сетка, варианты параллельно), типографикой (засечный TNR по умолчанию, sans, моноширинный), оформлением блоков задач (рамка, фон, боковая полоса, тонкая линия слева, без декора), палитрой (синий-фиолетовый Wildcat, охра, изумруд, лавандовый, бирюзовый, графитовый, бордовый акцент) и целевым профилем (математика 6–11 / информатика / контрольная / тренажёр). Полная таблица метаданных — в [`registry.json`](./registry.json) рядом.

## Как добавить новый шаблон

Создайте папку `T<N>/` и файл `T<N>.tex`. Подключите Wildcat-преамбулу и расширение для автопроверки:

```latex
\documentclass[a4paper,11pt]{article}
\usepackage{../../../../Lessons/_templates/preamble_worksheet}
\usepackage{../_shared/worksheet_check}
```

Переопределите цвет акцента в начале документа (если нужно): `\definecolor{wcprimary}{HTML}{4A5568}` (и при необходимости `wcprimaryLight`, `wcprimaryBG`). Затем при желании `\renewcommand{\TaskBox}[2]{...}` под свой дизайн карточек (или `\renewcommand{\WorksheetTitle}[1]{...}` для заголовка). Внутри `\begin{document}` используйте `\WorksheetTitle{...}`, `\FirstPageHeader`, далее свои задачи, в конце — `\WorksheetQR{<slug>/L1/v1}`. Не забудьте добавить запись в `registry.json`.

## Поле ответа и автопроверка

В каждой задаче место под ответ ставьте через `\AnswerField{N}` (где `N` — человекочитаемый идентификатор задачи, обычно номер). Эта команда из `_shared/worksheet_check.sty` рисует серую рамку и одновременно записывает координаты бокса в `.aux` через `\rlAFmark{...}` — это позволяет Python-парсеру РабочийЛист.ai точно знать, в какой области страницы ученик пишет ответ, и применять OCR/распознавание именно там. Размер бокса можно переопределить: `\SetAnswerFieldSize{30mm}{8mm}` (ширина, высота).

Все шаблоны компилируются через **xelatex** (нужен fontspec + Times New Roman/Arial из преамбулы Wildcat). Не используйте pdflatex.
