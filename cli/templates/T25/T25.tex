\TaskBox{I}{%
  При каких значениях параметра $a$ уравнение $x^2 - (a + 3)x + 3a = 0$ имеет два различных корня, оба больше $2$?\par
  \vspace{1.5mm}\hfill\textit{Ответ:}\ \AnswerField{1}%
}

\TaskBox{II}{%
  Найдите все натуральные $n$, при которых число $n^2 + n + 41$ не является простым, и в ответе укажите наименьшее такое $n$.\par
  \vspace{1.5mm}\hfill\textit{Ответ:}\ \AnswerField{2}%
}

\TaskBox{III}{%
  В правильной четырёхугольной пирамиде $SABCD$ длина бокового ребра равна $13$, длина стороны основания равна $10$. Найдите длину отрезка, соединяющего вершину $S$ с серединой ребра $AB$.\par
  \vspace{1.5mm}\hfill\textit{Ответ:}\ \AnswerField{3}%
}

\TaskBox{IV}{%
  Найдите наименьшее значение функции $f(x, y) = x^2 + y^2 - 4x - 6y + 18$ на множестве пар $(x, y)$, удовлетворяющих условию $x + y = 5$.\par
  \vspace{1.5mm}\hfill\textit{Ответ:}\ \AnswerField{4}%
}
